⚙️ ตั้งค่า Google Apps Script (ทำครั้งเดียว)

  1. เปิด Google Sheet → คลิกเมนู Extensions → Apps Script
  2. ลบโค้ดเดิมออกทั้งหมด แล้ววางโค้ดจากไฟล์ Code.gs ที่ได้รับ
  3. คลิก Deploy → New deployment → เลือก Web app
  4. ตั้ง Execute as: Me และ Who has access: Anyone
  5. คลิก Deploy → คัดลอก Web app URL ที่ได้
  6. วาง URL ด้านล่างแล้วกด บันทึกและโหลด
